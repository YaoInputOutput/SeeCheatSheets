# 插入模式
| 按键        | 作用                           |
| ----------- | ------------------------------ |
| `<C-A>`     | 行首                           |
| `<C-E>`     | 行尾                           |
| `<C-Left>`  | 左移一个单词                   |
| `<C-Right>` | 右移一个单词                   |
| `<S-Left>`  | 向左缩进                       |
| `<S-Right>` | 向右缩进                       |
| `<S-Down>`  | 将本行下移                     |
| `<S-Up>`    | 将本行上移                     |
| `<C-CR>`    | 下行插入                       |
| `<C-W>`     | 删除光标前单词                 |
| `<C-U>`     | 删除光标前文本，删除光标后文本 |
| `<C-K>`     | 删除光标后单词                 |
| `<C-L>`     | 删除光标后一个字符             |
| `<M-e>`     | 快速包围(fast wrap)            |

# 普通模式
* 复合操作符：
   `d` `c` `y` `gu` `gU` `g~` `v`
* 光标移动：
   `0` `^` `$` `w` `W` `b` `B` `e` `E` `ge` `gE`
   `f` `F` `%` `(` `)` `{` `}`
   `gg` `G` `H` `L` `M` `{num}|` `gm`
* 文本对象：需要带前缀`i`或`a`
   `w` `s` `p` `(` `{` `[` `<` `"` `'` `` ` ``
   `e` `l` `i` `f` `,`

| 按键        | 作用              |
| ----------- | ----------------- |
| `<C-A>`     | 行首              |
| `<C-E>`     | 行尾              |
| `<C-Left>`  | 左移一个单词      |
| `<C-Right>` | 右移一个单词      |
| `<S-Left>`  | 向左缩进          |
| `<S-Right>` | 向右缩进          |
| `<S-Down>`  | 将本行下移        |
| `<S-Up>`    | 将本行上移        |
| `[<Space>`  | 行前添加一空行    |
| `]<Space>`  | 行后添加一空行    |
| `r`         | 替换光标下字符    |
| `R`         | 进入替换模式      |
| `~`         | `g~l`             |
| `x`         | `dl`              |
| `s`         | `cl`              |
| `D`         | `d$               |
| `C`         | `c$`              |
| `dd`        | 删除当前行        |
| `cc`        | 删除当前行        |
| `.`         | 重复上次操作      |
| `u`         | undo(按栈序)      |
| `<C-R>`     | redo(按栈序)      |
| `g;`        | 跳转旧的修改处    |
| `g,`        | 跳转新的修改处    |
| `[c`        | 跳转上个diff处    |
| `]c`        | 跳转下个diff处    |
| `[e`        | 跳转上个problem处 |
| `]e`        | 跳转下个problem处 |
| `;{char}`   | 跳转字母char处    |

# 复制粘贴
| 按键        | 作用                     |
| ----------- | ------------------------ |
| `,`         | `yl`                     |
| `Y`         | `y$`                     |
| `<Leader>y` | `"+y`                    |
| `<Leader>,` | `"+yl`                   |
| `<Leader>Y` | `"+y$`                   |
| `<Space>y`  | 复制整个文件到系统剪切板 |
| `<Leader>p` | `"+p`                    |
| `<Leader>P` | `"+P`                    |
| `<Leader>o` | 粘贴寄存器+到下行        |
| `<Leader>O` | 粘贴寄存器+到上行        |
| `<Space>p`  | 粘贴系统覆盖至当前文件   |
| `=p`        | `"0p`                    |
| `=P`        | `"0P`                    |
| `=o`        | 粘贴寄存器0到下行        |
| `=O`        | 粘贴寄存器0到上行        |

# Editor窗口操作
| 按键        | 作用                                   |
| ----------- | -------------------------------------- |
| `<C-W><Up>` | 跳转上方的Editor Group，其他方向键同理 |
| `<C-W>K`    | 将Editor Group上移，其他方向键同理     |
| `<Tab>`     | 跳转下个Editor Group                   |
| `<S-Tab>`   | 跳转上个Editor Group                   |
| `<C-1>`     | 跳转指定Editor Group，其他数字同理     |
| `<C-Tab>`   | 切换Editor                             |
| `<C-W>v`    | 竖直切分                               |
| `<C-W>s`    | 水平切分                               |
| `<C-W>=`    | 均布Editor Group窗口大小               |
| `<C-W>c`    | 关闭当前Editor                         |
| `<C-W>o`    | 仅保留当前Editor                       |
| `<C-S-T>`   | 重开关闭的Editor                       |

# 折叠操作
| 按键     | 作用             |
| -------- | ---------------- |
| `<C-Z>z` | 切换（所有）折叠 |
| `<C-Z>r` | 打开（所有）折叠 |
| `<C-Z>m` | 关闭（所有）折叠 |
| `<C-Z>n` | 跳转下一个折叠   |
| `<C-Z>b` | 跳转上一个折叠   |
| `<C-Z>p` | 跳转上级折叠     |

# 书签标记
| 按键     | 作用               |
| -------- | ------------------ |
| `<C-M>m` | 切换标签状态       |
| `<C-M>i` | 为标签添加注释     |
| `<C-M>n` | 跳转至下一个标签   |
| `<C-M>b` | 跳转至上一个标签   |
| `<C-M>l` | 列出当前文件的标签 |
| `<C-M>L` | 列出所有标签       |
| `<C-M>c` | 删除当前文件的标签 |
| `<C-M>C` | 删除所有标签       |

# 代码编辑
| 按键                 | 作用                           |
| -------------------- | ------------------------------ |
| `<Tab>`              | 下个补全项，或下个代码片段锚点 |
| `<S-Tab>`            | 上个补全项，或上个代码片段锚点 |
| `<CR>`               | 选中补全项                     |
| `<Tab>`              | 根据snippet前缀补全代码片段    |
| `<A-/>`              | 搜索snippet                    |
| `<C-/>`              | 注释代码                       |
| `cs{char}{char}`     | 替换包围符                     |
| `ds{char}{char}`     | 删除包围符                     |
| `ys{text-obj}{char}` | 添加包围符                     |
| `yss{char}`          | 为改行添加包围符               |
| `<C-A-Up>`           | 向上添加多光标                 |
| `<C-A-Down>`         | 向下添加多光标                 |
| `<C-N>`              | 选中下个匹配单词并添加光标     |
| `<C-S-L>`            | 所有匹配当前单词的地方添加光标 |
| `<C-LeftMouse>`      | 添加多光标                     |
